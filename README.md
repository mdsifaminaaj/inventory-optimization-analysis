# Inventory Optimization & Sales Analysis

## 📌 Problem

Analyze retail sales, purchase, and inventory data to optimize stock levels and reduce inefficiencies.

## 📊 Key Analysis

* Demand Forecasting (ARIMA)
* ABC Analysis
* EOQ & Reorder Point
* Lead Time Analysis
* Dead Stock Detection

## 🔍 Key Insights

* 20% products contribute ~70% revenue (A-category)
* Demand increases during weekends
* Supplier B has highest lead time → risk of stockouts
* Some products show zero sales → dead stock

## 💡 Business Impact

* Reduced stockout risk
* Improved inventory planning
* Better supplier decision-making

## ▶️ How to Run

pip install -r requirements.txt
jupyter notebook notebooks/analysis.ipynb
